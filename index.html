hexo+git部署个人博客

Posted on 2020-12-05
环境部署

1、安装git

ssh config –global user.name “skukooo”

ssh config –global user.email “xg050337@163.com“

ssh-keygen -t rsa生成密钥和公钥，将公钥中的内容复制到github的SSH keys中

通过ssh -T git@github.com测试是否绑定到了github

2、安装nodejs
在系统变量中新建：

变量名：NODE_PATH
变量值：自定义的模块安装的路径

修改用户变量中的path：

npm路径改为：nodejs/node_global

3、安装hexo（利用nmp）

更改nmp镜像源：

npm config set registry http://registry.npm.taobao.org

初次安装需要hexo init和npm install

hexo g 生成页面

hexo s 启动预览

4、部署hexo到github pages

npm install hexo-deployer-git –save

修改 _config.yml的deployment部分
type：git
repository：git@github.com：skukooo/skukooo.github.io.git
branch：master

运行hexo d

5、以后每次写完运行hexo g和hexo d即可
