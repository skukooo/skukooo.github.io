转行

Posted on 2020-12-09 Edited on 2020-12-24 Valine:
工作各项对比
送变电土建 前端开发
是央企，稳定，在疫情到来的时候尤为明显 有裁员的风险，年龄危机（存疑）
工资待遇、福利还行 待遇方面差不多
管理路线主要是在分包、设计、业主、监理之间周旋 将UI设计和产品经理的想法转化成代码
技术路线主要是写方案、审方案 后期可以前后端打通成为全栈工程师
有很多应酬（饭局、酒局），分包单位、领导检察 基本无应酬
工作地点不定，野外作业多 工作地点相对固定
为什么想转行

1、工作内容完全没兴趣。技术方面，不是个人喜好的那种不喜欢，从下面两种情况可以体现：①解决问题后没有成就感；②学不进去；这两种情况在土木施工这一块体现的很明显，而且两者相辅相成。与之相对的，计算机前端（当然也包括后端，实际上是计算机的绝大部分领域）的东西，我解决一个哪怕很小的问题，都会有成就感，这种正反馈对于持续地深入地钻研是非常重要的，如果长期没有正反馈，人会变得颓废、自闭、不思进取，至少我是这样的。
2、管理方面，送变电应酬很多，酒桌文化极为浓厚，不适应，适应了这么久，还是不适应，这与我倾向于静静地做技术的性格格格不入。
3、设计院是否是一个可选的方向，设计院倒是可以静静地做技术，但还是不适合，具体参考第1条。
4、对于丝毫不感兴趣的事，遇到挫折容易消极，因为没有积极面对，然后解决问题的动力。
5、东奔西走，很不方便，各方面都是，不过这是次要原因，上面四条关乎未来职业发展，是核心问题。
总结：也许任何事情，做久了都会厌，但是至少刚开始的十年应该是充满热情和冲劲的，如果刚开始就很消极很颓废，那就没有未来可言了，总的来说，送变电的工作氛围和我几乎处处犯冲。现在每天早上起来，心情会瞬间跌至低谷，整个人像游魂一样。每天心情最好的时候就是能够睡觉前能看一会儿计算机前端的内容，这已经成了呆在工地而不崩溃的唯一一个支撑点了。工作可以让人觉得苦，但不能让人觉得痛苦
这不是一时兴起，而是初心不改

转行穷三年，具体计划

要有转行穷三年的觉悟，要提前有详细的规划
目标单位：字节跳动、腾讯
目标岗位：前端开发
自身劣势：非科班出身，数据结构、算法、操作系统原理等基础课程学得不够深入（和科班出身的优秀人物相比而言）

招聘网站上所列岗位要求

1、精通HTML/JS/CSS
2、对主流前端框架有一定了解，并至少对其中一种有深入了解
3、熟悉 JS、NodeJS、Chromium、Electron 至少一种
4、熟练掌握 SVG / Canvas / WebGL 等前端绘图技术及标准
5、数据可视化类库 D3 / G2 / Three.js / ECharts / Highcharts
